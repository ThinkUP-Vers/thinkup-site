# Pack A + C — Think’UP GitHub Pages

Contenu fourni :
- Pages piliers (A) : index-iceberg, taches-invisibles, diagnostic-roi-ia, gains-rapides-ia, cadrage-adoption-ia
- Page hub (C) : ressources.html
- Maillage interne : ajout du lien "Ressources" dans la navigation + footer (toutes pages)
- sitemap.xml (à soumettre dans Google Search Console)

Déploiement :
1) Copier tous les fichiers dans la racine du repo GitHub Pages (même niveau que index.html).
2) Vérifier que styles.css est bien à la racine (déjà le cas sur le site).
3) Pousser (commit + push). GitHub Pages publie automatiquement.
4) Google Search Console : ajouter et soumettre le sitemap : https://thinkup-vers.github.io/thinkup-site/sitemap.xml

Remarque :
- robots.txt à la racine du domaine est préférable, mais GitHub Pages sous chemin /thinkup-site limite cette possibilité sans domaine custom.
