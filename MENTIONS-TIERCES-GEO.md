# Kit mentions tierces — visibilité GEO de Think'UP

Les études 2025-2026 convergent : le facteur le mieux corrélé à la présence
d'une marque dans les réponses des IA (ChatGPT, Perplexity, AI Overviews)
est le volume de **mentions sur des sites tiers crédibles** — loin devant
les backlinks classiques. Voici les actions par ordre d'impact/effort,
toutes à faire une seule fois sauf mention contraire.

## 1. Référencements institutionnels (gratuits, forte autorité)

- [ ] **Activateur France Num** — LE référencement prioritaire pour un
      consultant numérique ciblant les PME. Dossier en ligne :
      https://www.francenum.gouv.fr/activateurs (« Devenir activateur »).
      Bénéfice : fiche publique sur un domaine gouv.fr, très repris par
      les IA pour les requêtes « accompagnement numérique PME ».
- [ ] **Annuaire CCI Versailles-Yvelines** : se déclarer comme prestataire
      de conseil auprès de la CCI (et demander l'inscription aux annuaires
      de prestataires transformation numérique s'ils existent).
- [ ] **Bpifrance** : si éligible, se référencer parmi les consultants
      pouvant intervenir sur les Diag Data/IA — renseignements via le
      conseiller Bpifrance régional. Même sans référencement formel,
      figurer dans les listes régionales de prestataires IA aide.

## 2. Fiches et profils à aligner (cohérence d'entité)

Toutes les fiches doivent utiliser EXACTEMENT : **Think'UP** (avec
l'apostrophe), think-up.fr, patrick@think-up.fr, 22 rue Victor Hugo,
78000 Versailles. La cohérence nom/adresse/téléphone est un signal
d'entité majeur pour les IA.

- [ ] **Google Business Profile** : vérifier/compléter la fiche (catégorie
      « Consultant », zone Île-de-France, lien vers think-up.fr et la
      boutique).
- [ ] **LinkedIn** : page entreprise Think'UP alignée (description reprenant
      la formulation du llms.txt : conseil en adoption de l'IA, méthode
      Index Iceberg, ROI en 90 jours) + profil personnel pointant vers
      think-up.fr.
- [ ] **Pages Jaunes / annuaires locaux 78** : une fiche propre suffit.

## 3. Presse et contenus tiers (récurrent, le plus puissant)

- [ ] **Presse locale éco** (78actu, La Gazette SQY, Le Parisien 78 éco) :
      proposer un angle « un consultant versaillais aide les PME à mesurer
      le ROI de l'IA avant d'investir » — les IA citent massivement la
      presse régionale pour les requêtes localisées.
- [ ] **Médias PME nationaux** (Chef d'entreprise, Les Echos Entrepreneurs,
      BFM Business PME) : tribunes d'expert. Un angle fort déjà prêt :
      « L'Iceberg Index du MIT mesure l'économie américaine ; voici comment
      appliquer la même image à VOTRE PME » — capitalise sur la notoriété
      de l'étude MIT au lieu de la subir.
- [ ] **Podcasts / webinaires PME** : chaque participation = une page tierce
      qui cite Think'UP.
- [ ] **Favikon Top 20 IA France 2025** : mention déjà acquise — la citer
      systématiquement dans les bios et communiqués (elle est déjà dans le
      llms.txt, c'est le bon réflexe).

## 4. Signaux techniques complémentaires

- [ ] **Bing Webmaster Tools** : inscrire think-up.fr (l'index Bing alimente
      ChatGPT Search et Copilot). Import possible depuis Google Search
      Console en 2 clics : https://www.bing.com/webmasters
- [ ] **IndexNow** : activable depuis Bing Webmaster Tools pour accélérer
      l'indexation des nouvelles pages (articles du blog notamment).
- [ ] **Cadence éditoriale** : 1 article/mois minimum — 85 % des citations
      des IA pointent vers des contenus de moins de 2 ans ; la fraîcheur
      est un critère mesuré.

## Suivi

Re-tester tous les 2 mois la visibilité sur les requêtes cibles
(« consultant IA PME France », « mesurer ROI IA PME », « accompagnement
IA dirigeants ») dans ChatGPT, Perplexity et Google AI Overviews, et
noter si think-up.fr apparaît dans les sources citées.
